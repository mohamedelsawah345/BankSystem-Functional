# 🏦 Bank Management System (Functional Paradigm)

A secure, terminal-based banking application developed using the Functional Programming paradigm in C++. This project was built as a comparative study to master state management, procedural logic, and data flow before transitioning to Object-Oriented architectures.

## 🚀 Key Technical Highlights

* **Stateless Logic:** Focused on pure functions and structured data (`struct`s) to manage bank operations without complex class hierarchies.
* **Data Flow Management:** Implemented robust file-based persistence using procedural parsing and serialization techniques.
* **STL Integration:** Heavy use of C++ Standard Template Library (STL) algorithms and vectors for efficient data handling.
* **Comparative Evolution:** This project serves as the direct precursor to my [OOP-based Bank System](https://github.com/mohamedelsawah345/Bank-Broject-Using-OOP), demonstrating my ability to refactor and evolve codebases into modern architectures.

## 🛠️ Tech Stack

* **Language:** C++
* **Paradigm:** Functional / Procedural Programming
* **Storage:** Flat-file database (CSV-style)
